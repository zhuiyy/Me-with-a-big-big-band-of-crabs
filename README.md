为了能令人一眼笑出来，最好把精华，抽象的，哪怕只有一句话放进README.md里，例如：
  
  这篇README.md中至少包含三个‘README.md’! Lets find them out!(GEB)
如你所见，GEB，记忆笔，加油吧，进一步等笑话最优先，优先上有更优先，wow！GEB！

tips：GEB = Godel, Escher, Bach-- An Eternal Golden Braid
